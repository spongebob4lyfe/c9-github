Programming Software and Language

Overall, my Generative Art page functions with HTML, which is referenced to a Javascript file, a coding programming language that is most commonly used to create interactive art within web browsers. This allows the programmer to type up commands and code that the computer can translate into a language that it understands. Basically, a coding, or programming, language is a series of syntax rules that can explain how code should be written and presented. The language a computer uses and understands is binary code, which is code made up of combinations of 0s and 1s that would take eons and an extremely advanced human mind to write out. Therefore, we use several different coding languages that we understand and can easily create, with certain languages made for certain purposes. When we write a certain command, it is first translated into assembly language, which is a low-level programming language for a computer, meaning it is closer to the binary language a computer understands, unlike the high-level language mostly used today. Then, the assembly language is translated into machine language, a set of instructions that is executed directly by the CPU, the computer's central processing unit, into binary code so that the computer can understand the code.

These actions are made possible with the use of Javascript functions, which is a block of code that performs a certain action or represents a certain value using commands, conditions, keywords, parameters, and more.

Code Specifics

Javascript Functions and Commands

Ellipses

The ellipse command creates ellipses, or circles and ovals, in the display window. They are one of the most common shapes used to create other figures and art, like I did for my generative art when making the potatoes and rocks. To call the ellipse function, write the word ellipse, followed by a set of parentheses, which contain the parameters. The first parameter represents the x value of the center point of the ellipse, and the second parameter represents the y value of the center point of the ellipse. The third parameter represents the width of the ellipse and the fourth represents the height. To create a perfect circle, both the third and fourth parameters, the width and height, must be equal.

For Loops

The for loop function allows commands to repeat a certain amount of times. To call a for loop, type the word for, followed by a set of parentheses. In the parentheses are three parts separated by semicolons: the initialization, the conditional, and the update. The initialization defines the variable, the conditional controls how many times the loop is executed, and the update represents the increment or decrement. After this is the code block consisting of the statements that are to be looped in the curly brackets. The loop continues until the conditional executes to be false. When the conditional is true, however, then the following statements are executed and looped.

Conditional Statements

Conditional statements, also known as if-else statements, allow the program to execute blocks of code under certain cicumstances. To call an if-else statement, type the word if, followed by a set of parentheses containing a condition, which is a statement that must be true or false, like x > 20, for example. After this is a code block, which only runs when the condition is true. If the condition is false, then the codeblock will not run. However, if the conditional statement has an else part as well, then the codeblock in curly brackets following the else() will be executed.

Functions

You can make your own functions to be able to rewrite the function more than once without having to write all of the code again. This can be done by creating commands and properties that come together to create something. Then, if you want to duplicate these commands to make a copy of the creation on the display screen, you can put it into a function by calling it with a keyword that appropriately represents it and add = function, followed by a set of parentheses and curly brackets, which contain the commands. Finally, to repeat this entire function, you just have to type the keyword and a set of parentheses and a semicolon following it.

Interactive MouseClicked Function

First, in the parameters of the function that will be put into the mouseClicked function, replace all of the x and y coordinates with expressions that are related to a certain starting number, but also equal the original number. For example, if the number is 30, then it must be changed to 200-70, with all of the others changed to be related to that same number, 200, as well. To write out the mouseClicked function, which allows us to make the function appear on the display screen wherever the mouse is clicked each time, write the keyword mouseClicked, followed by = function( ){ }. Then, put the function that was typed out previously into the curly brackets and type mouseX, mouseY into the parentheses of the rewritten function and x, y into the original function containing the commands. This means that all of the 200s, or whatever starting number used in the parameters of the commands, must be replaced with x or y, depending on which variable they represented, allowing the function to appear anywhere on the web page when it is clicked.

Programming Decisions

My Potato Function

For my function, I decided on creating a potato. I did this by using ellipses for the overall shape, eyes, and dots and by using arcs for the smile and outlines. I had to use arcs for the outline instead of stroke and strokeWeight because the ellipses had to overlap to create the potato figure, and if I used stroke, there would be unwanted lines across the potato. I decided on using potatoes because they are delicious and I just overall enjoy them, like the way they look and how the word potato sounds. I also decided on putting a smiley face on the potato because it represents potatoes make me feel good inside and also because happy potatoes are just better in general.

My For Loops

I used for loops when creating rocks in the ground and the sky. I originally was just going to use it for the rocks, but it ended up being a part of the sky as well only because it was required, as one of my peers showed me. I decided on making them rocks because it makes the dirt seem more realistic and detailed, which makes the overall art better. Using a for loop for the rocks was an easy and efficient way of creating several various shaped rocks at once, as it would be a waste of time to create that many ellipses and making the rocks all the same shape would make them seem fake and less detailed.

My Conditional Statements

For my conditional statements, I ended up creating several of them, since I was attempting to make a sky with more than one shade of blue. I did this with many else if statements, with each one of them representing a part of the skay. To be more specific, each statement represents a rectangualar piece of the sky, in which there are a total of five. I made it lighter by subtracting 2 from both the green and blue parameters of the fill property, which had already set variables as parameters. I wanted the sky to appear lighter as it went down, and using conditional statements was an effective way of create detailed and realistic skies, which I was taught by one of my classmates.
